# plastid_stats
English:
Plastid_stats is a Python-based tool developed using the Biopython library to assess the presence of plastid sequences within metagenomic bins. The tool requires as input a FASTA file containing contigs classified as plastids by classifiers such as Tiara or CORGI, which assign specific plastid-related identifiers to sequences. Plastid_stats compares these plastid-classified contig IDs with the contig IDs present in each bin, allowing the quantification of plastid content per bin. The results include both the number and the proportion of plastid-associated contigs and nucleotides, enabling the identification of bins that are likely of plastid origin.

Portuguese:
Plastid_stats é uma ferramenta desenvolvida em Python utilizando a biblioteca Biopython para avaliar a presença de sequências plastidiais em bins metagenômicos. A ferramenta utiliza como entrada um arquivo FASTA contendo contigs classificados como plastídios por classificadores como Tiara ou CORGI, que atribuem identificadores específicos relacionados a plastídios às sequências. O plastid_stats compara esses IDs de contigs classificados como plastidiais com os IDs presentes em cada bin, permitindo a quantificação do conteúdo plastidial por bin. Os resultados incluem tanto o número quanto a proporção de contigs e nucleotídeos associados a plastídios, possibilitando a identificação de bins com provável origem plastidial.
